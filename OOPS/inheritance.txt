inheritance means acquiring the properties and behaviors of the parent class is called as inheritance
it is done with the help of extends keyword.

representation
child->parent IS-A relationship
child is a parent
like wise dog is a Animal

the child can have other properties and behaviors along with the properties and behaviours of the parent class.
 the parent can be used to refer the child object.
 i,e parent can access the inherited ,overridden methods in the childclass when we create the object with the parent reference.

 the parent reference cannot access the specialized of the child class.

 the parent of every class is Object.
 the private variable will not be accessed into the child class. because if they are accessed then it voilates the the property of encapsulation.
============================================================================
SUPER()

the super is used to call the parent class, the super is used to access the variable , methods,constructors of the parent class in the child class.

we should know the other things such as upcasting and downcasting.
upcasting -> when the parent refernce is used to create the child object. this is called as upcasting
now with the parent reference when we want to access the specialized methods of the child class . then we need to do the downcasting.
class Animal{
	int age;
	String name;
	public int getAge() {
		return age;
	}
	public void setAge(int age) {
		this.age = age;
	}
	public String getName() {
		return name;
	}
	public void setName(String name) {
		this.name = name;
	}
	
	void sleep() {
		System.out.println("the "+name+" is sleeping");
	}
	
}
class Dog extends Animal{
	//here all the properties and behaviors of the parent class Animal are present here , animal is the parent then dog is the child.
	//String name="Shepard";
	
	void run() {
			System.out.println(name +" is running");
	}
	
}

public class Inheritance {

	public static void main(String[] args) {
		// TODO Auto-generated method stub
		Animal a1=new Animal();
		a1.setName("Parent Animal");
	 Animal a=new Dog();// this is called as Upcasting.
	 //a.run();// run() is a specialized method. so the parent class reference cannot access the run
	 a.setName("Animal is Animal");
	 a.sleep();
	 ((Dog)a).run();// this is called as downcasting.
	 
	}

}

output:
the Animal is Animal is sleeping
Animal is Animal is running

		return name;

		this.name = name;
	}

		System.out.println("the "+name+" is sleeping in parent class");
	}
	
}
class Dog extends Animal{
	//here all the properties and behaviors of the parent class Animal are present here , animal is the parent then dog is the child.
	//String name="Shepard";

		System.out.println("this is dog");

}
